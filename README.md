# Leiden Data Stewardship Wizard (DS Wizard) User Guide

Welcome to the Leiden DS Wizard (https://leiden.ds-wizard.org/) User Guide, which is there to help you create and maintain a Leiden DMP.
The Leiden DMP available in the Leiden DS Wizard is a (pilot) adaptation of the Leiden DMP available in Zenodo (https://doi.org/10.5281/zenodo.3903267).

## Usage

Go to https://lacdr.github.io/dsw-guide/ to visit the guide, and select the *Create DMP* or *Review DMP* tabs at the top.

## Contribute to this guide

Contributions to this guide are very welcome. On every page in the guide there are convenient buttons to report issues or to propose edits. This guide is developed as a Quarto site (https://quarto.org/docs/websites/), and existing pages can easily be added using Markdown (https://quarto.org/docs/authoring/markdown-basics.html).
